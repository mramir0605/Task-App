# Project 7 - *Task App*

Submitted by: **Mary Ramirez**

**Name of your app** is an app that allows users to stay organized by creating and managing a personalized task list. Tasks can be added, marked as completed, edited, or deleted with ease. All task data is saved locally using UserDefaults, ensuring that tasks persist even after the app is closed. The app also includes a calendar view, giving users a clear way to plan and visualize upcoming tasks.

Time spent: **2** hours spent in total
